# Introducción al problema

En este proyecto resolvemos la siguiente integral utilizando el método numérico de Cuadratura Gaussiana:

$$ I = \int_{0}^{\pi} dx \sin(x^2) $$

Este es un caso clásico donde los métodos computacionales son indispensables, dado que esta integral no posee una solución analítica mediante funciones elementales. A lo largo de esta documentación, se explicará el método, cómo utilizar el código y se detallará cada función implementada.
